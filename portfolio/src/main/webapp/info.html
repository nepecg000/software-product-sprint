AggiePass
A platform provides TAMU students either has a NCAA football season tickets but not intend to
watch, or those who don't have tickets but want to watch it, it will serve to ease the process of
buying or selling between them. Through this application, buyers can filter out which game they
want to watch, and the price. And arrange a meetup time to exchange passes and complete the
transaction. We host a PHP server in the back-end to deal with our database.
VibeCheck
A jacket which has a camera and a Raspberry Pi inside, aimed to help people who have a visual
impairment to detect objects. It uses Darkflow (a Python module implement darknet with TensorFlow)
and YOLO (You Only Look Once) as the model to detect objects.

https://pjreddie.com/darknet/yolov1/
